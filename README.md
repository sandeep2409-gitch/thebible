# పరిశుద్ధ గ్రంథము - తెలుగు బైబిల్ PWA (Telugu Bible)

పరిశుద్ధ గ్రంథము (Telugu Bible PWA) అనేది మొబైల్ నిష్పత్తి (Mobile Aspect Ratio) ఆధారంగా ప్రత్యేకంగా రూపొందించబడిన ఒక ప్రీమియం ప్రోగ్రెసివ్ వెబ్ అప్లికేషన్ (Progressive Web App). ఇది 100% ఆఫ్‌లైన్ వినియోగం, వాయిస్ పఠనం (Offline Text-to-Speech), వేగవంతమైన శోధన, హైలైట్లు, నోట్స్ మరియు బ్యాకప్ సౌకర్యాలతో కూడిన ఒక సంపూర్ణ నోట్‌బుక్ శైలి (Moleskine Ruled Notebook Style) గ్రంథ పఠన అనుభవాన్ని అందిస్తుంది.

యాప్ మొత్తం ఇంటర్‌ఫేస్ మరియు ఫీచర్లు పూర్తిగా **తెలుగు భాషలో** మాత్రమే రూపొందించబడ్డాయి.

---

## 🌟 ముఖ్య ఫీచర్లు (Key Features)

### 1. నోట్‌బుక్ శైలి అద్భుత రూపకల్పన (Notebook Ruled Aesthetics)
*   **ivory పేపర్ బ్యాక్‌గ్రౌండ్**: చదవడానికి కంటికి ఆహ్లాదకరంగా ఉండే క్లాసిక్ ఐవరీ కాగితం రంగు నేపథ్యంలో యాప్ డిజైన్ చేయబడింది.
*   **లైన్ల రూలింగ్ (Ruled Lines)**: పుస్తక కాగితాలపై ఉండే విధంగా క్లాసిక్ బ్లూ గీతలను బ్యాక్‌గ్రౌండ్‌గా అమర్చడం జరిగింది. ఈ లైన్లు టెక్స్ట్ పరిమాణం పెంచినప్పుడు దానంతటదే సర్దుబాటు అయ్యే విధంగా గణిత శాస్త్ర నియమాలతో (`1.9em`) రూపాకారం చెందాయి.
*   **ఎడమ మార్జిన్ రేఖ (Left Margin Red Line)**: నోట్‌బుక్స్‌లో ఉండే విధంగా ఎడమ వైపున గులాబీ రంగు మార్జిన్ రేఖ ఉంటుంది.
*   **మార్జిన్ వెలుపల సంఖ్యలు**: మార్జిన్ రేఖకు ఎడమ వైపున వర్స్ సంఖ్యలు, కుడి వైపున పవిత్ర గ్రంథ వాక్యాలు ఖచ్చితమైన సమలేఖనంతో (Alignment) అమర్చబడ్డాయి.
*   **లెదర్ కవర్ సైడ్‌బార్**: డెస్క్‌టాప్ కవర్ మరియు సైడ్ ప్యానెల్స్ ఒక ప్రీమియం లెదర్ బైండింగ్ మరియు కార్డ్‌బోర్డ్ క్రాఫ్ట్ ఆకృతిని కలిగి ఉంటాయి.

### 2. ఆఫ్‌లైన్ స్పీచ్ సింథసిస్ పఠనం (Speech Synthesis Audio)
*   **ఆఫ్‌లైన్ వాయిస్**: బ్రౌజర్ యొక్క అంతర్నిర్మిత HTML5 SpeechSynthesis ఇంజన్‌ను ఉపయోగించుకోవడం ద్వారా ఎటువంటి ఇంటర్నెట్ లేదా ఆడియో ఫైల్స్ డౌన్‌లోడ్ చేయకుండానే 100% ఆఫ్‌లైన్‌లో తెలుగు వాక్యాలను వినవచ్చు.
*   **లైవ్ వర్స్ ట్రాకింగ్**: ఆడియో చదువుతున్నప్పుడు ఏ వాక్యం చదువుతుందో ఆ వాక్యానికి ఆటోమేటిక్‌గా ఒక అందమైన గోల్డెన్ బార్డర్ హైలైట్ జోడించబడి, అది పేజీ మధ్యలోకి స్క్రోల్ అవుతుంది.
*   **వేగ నియంత్రణ**: చదివే వేగాన్ని `0.8x`, `1.0x`, `1.2x`, `1.5x` పరిమాణాలలో మార్చుకోవచ్చు.

### 3. సంపూర్ణ ఆఫ్‌లైన్ నిల్వ & బూట్ (100% Offline Resilience)
*   **ఆటోమేటిక్ ప్రొవిజనింగ్**: మొదటి సారి అప్లికేషన్ ఓపెన్ అయిన వెంటనే ఒక కస్టమ్ సర్వీస్ వర్కర్ (`sw.js`)static సైట్ షెల్, గూగుల్ ఫాంట్స్తో పాటు బైబిల్ యొక్క మొత్తం **66 పుస్తకాల JSON డేటాబేస్ (~12MB)** ను ఆటోమేటిక్‌గా బ్రౌజర్ కాష్‌లోకి డౌన్‌లోడ్ చేస్తుంది.
*   **జీరో డేటా మోడ్**: ఒకసారి లోడ్ అయిన తర్వాత, ఇంటర్నెట్ కనెక్షన్ పూర్తిగా లేకపోయినా మొత్తం బైబిల్ చదవడం, శోధించడం, ప్లే చేయడం మరియు నోట్స్ రాసుకోవడం వంటి ప్రతి ఫీచర్ నిరంతరాయంగా పనిచేస్తుంది.

### 4. గ్లోబల్ ఆఫ్‌లైన్ శోధన (Instant Global Search)
*   మొత్తం 66 పుస్తకాలలో (పాత నిబంధన మరియు క్రొత్త నిబంధన) ఏ పదాన్నైనా (ఉదా: కాపరి, ప్రేమ, దేవుడు) క్షణాల్లో వెతకవచ్చు.
*   శోధించిన పదాలు ఫలితాలలో హైలైట్ చేయబడతాయి మరియు దానిపై క్లిక్ చేయడం ద్వారా నేరుగా ఆ పుస్తకం యొక్క నిర్దిష్ట అధ్యాయంలోకి ప్రవేశించవచ్చు.

### 5. పర్సనల్ డైరీ హబ్ (Personalized Notebook Hub)
*   **రంగుల హైలైట్లు**: ముఖ్యమైన వాక్యాలను 4 రకాల పెస్టల్ రంగులతో (పసుపు, ఆకుపచ్చ, నీలం, గులాబీ) హైలైట్ చేసుకోవచ్చు.
*   **బుక్‌మార్క్‌లు**: త్వరగా చదవడానికి వాక్యాలను బుక్‌మార్క్ చేసుకోవచ్చు.
*   **వ్యాఖ్యానాలు/నోట్స్**: ప్రతి వాక్యం క్రింద మీ సొంత ఆధ్యాత్మిక ఆలోచనలను నోట్స్ రూపంలో రాసుకోవచ్చు.
*   **బ్యాకప్ మరియు రీస్టోర్**: మీరు సృష్టించిన మొత్తం సమాచారాన్ని (Highlights, Notes, Bookmarks) ఒకే ఒక్క `.json` ఫైల్‌గా డౌన్‌లోడ్ చేసుకుని భద్రపరుచుకోవచ్చు లేదా మరొక పరికరంలోకి లోడ్ చేసుకోవచ్చు.

### 6. మొబైల్ స్పెక్ట్ రేషియో అనుకరణ (Smartphone Bezel Simulator)
*   డెస్క్‌టాప్ లేదా లాప్‌టాప్‌లలో చూసేటప్పుడు, ఇది స్క్రీన్ మధ్యలో ఒక అందమైన స్మార్ట్‌ఫోన్ ఫ్రేమ్‌ను (`412px` x `892px`) ప్రదర్శిస్తుంది.
*   నిజమైన మొబైల్ ఫోన్లు లేదా టాబ్లెట్లలో ఓపెన్ చేసినప్పుడు, ఈ ఫ్రేమ్ తొలగించబడి స్క్రీన్ మొత్తం నిండుగా ఆటోమేటిక్‌గా విస్తరిస్తుంది.

## 🚀 ఎలా ప్రారంభించాలి (Getting Started)

స్థానికంగా సర్వర్‌ను రన్ చేసి యాప్‌ను ఓపెన్ చేయడానికి ఈ క్రింది విధానాన్ని అనుసరించండి:

### 1. అప్లికేషన్ హోస్ట్ చేయండి
యాప్‌లోని సర్వీస్ వర్కర్ మరియు AJAX ఫెచెస్ విజయవంతంగా పనిచేయడానికి దీనిని ఒక వెబ్ సర్వర్ ద్వారా రన్ చేయాలి. మీ టెర్మినల్/కమాండ్ ప్రాంప్ట్‌లో ప్రాజెక్ట్ ఫోల్డర్‌కు వెళ్లి ఈ క్రింది కమాండ్స్ రన్ చేయండి:

**పైథాన్ (Python) ద్వారా:**
```bash
python3 -m http.server 8080
```

**నోడ్ (NodeJS / npm) ద్వారా:**
```bash
npx http-server -p 8080
```

### 2. బ్రౌజర్‌లో ఓపెన్ చేయండి
సర్వర్ రన్ అయిన తర్వాత, బ్రౌజర్ ఓపెన్ చేసి ఈ క్రింది లింక్ ని సందర్శించండి:
👉 **[http://localhost:8080](http://localhost:8080)**

## 🛠️ డెవలపర్ చిట్కా: కాష్ క్లియర్ చేయడం (Bypassing PWA Cache)

యాప్ ఫైళ్లలో ఏవైనా మార్పులు చేసినప్పుడు, సర్వీస్ వర్కర్ పాత కాష్ ఫైల్స్ ని లోడ్ చేయకుండా ఉండడానికి ఈ క్రింది పద్ధతులను ఉపయోగించండి:

1.  **హార్డ్ రిఫ్రెష్ (Hard Refresh)**:
    *   **Mac (Chrome, Safari, Firefox)**: `Cmd + Shift + R` ప్రెస్ చేయండి (లేదా `Shift` ప్రెస్ చేసి రీలోడ్ బటన్ క్లిక్ చేయండి).
    *   **Windows (Chrome, Edge)**: `Ctrl + F5` ప్రెస్ చేయండి.
2.  **ఇన్‌కాగ్నిటో విండో**: కొత్త ఇన్‌కాగ్నిటో (ప్రైవేట్) విండోలో యాప్‌ను ఓపెన్ చేయడం ద్వారా బ్రౌజర్ కొత్త లేఅవుట్ ని త్వరగా అప్‌డేట్ చేస్తుంది.
3.  **DevTools ద్వారా**: బ్రౌజర్‌లో `Inspect` క్లిక్ చేసి -> **Application** ట్యాబ్ -> **Storage** సెలెక్ట్ చేసి -> **Clear Site Data** క్లిక్ చేసి పేజీని రీలోడ్ చేయండి.
